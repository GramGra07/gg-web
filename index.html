Project Details

ZipList: This is a flutter application built for iOS and web that allows you to create a very simple and minimalistic todo list. The idea came from me wanting to have a todo list, but all of the ones I found were expensive. My idea was to create the perfect todo list for me and everyone else.

FTC-Scouting Application: This is an application I built with Flutter that allows FTC teams to track matches locally on their phones and analyze data from the app.

OSLC Application: This is an application I built with Flutter that allows members of the church to connect more with the church.

Code Training Module: This is a repository I built in order to help the entire FTC community with the coding side of robotics. It provides useful code examples, as well as walkthroughs to get teams started and help them continue further in FTC Java coding.

RE Capstone Project: Final project for Robotics Engineering class completely coded in C++.

FTC Scouting Logger: This was created in order to help teams scout at their competitions and collect data on teams they would potentially be partnered with. It is completely customizable and created in Python.
